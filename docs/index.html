关于立冷科技

专业的液冷技术团队

立冷科技由香港理工大学王钻开教授团队创立，专注于数据中心液冷解决方案的研发与应用。

我们致力于为客户提供高效节能的热管理技术，通过创新的结构热铠甲技术，实现数据中心的智能化温控管理。

我们的解决方案具有高能效比(PUE < 1.06)、高水效比(WUE < 0.5)和低碳排放的特点。
